Arduino Lesson

Goals ordered by lesson

  1. Install Arduino development environment and configuration
  2. Program technical ( coding style )
  3. General Purpose Input and Output ( GPIO )
  4. LCD Module Control ( text mode )
  5. Communication to PC ( via Arduino IDE )
  6. Communication via Bluetooth
  7. Communication to other Systems ( UART, SPI, I2C )
  8. Analog and Digital conversion ( read variable voltage )
  9. Motor control ( DC motor with H bridge )
  10. Motor control ( Servo motor - PWM )
